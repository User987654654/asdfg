Описание
--------
Система управления задачами позволяет создавать и отслеживать задачи, назначать исполнителей
и управлять статусами выполнения. Система поддерживает различные приоритеты задач и
предоставляет гибкие возможности для управления рабочим процессом.

Основные возможности
-------------------
* Управление задачами
* Назначение приоритетов
* Отслеживание статусов
* Работа с исполнителями
